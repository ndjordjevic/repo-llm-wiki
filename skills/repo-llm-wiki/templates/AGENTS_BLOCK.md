## Repo wiki

This repo has a generated wiki in `wiki/`. Before answering questions about
the repo's shape, components, or deployment, read `wiki/index.md` and follow
[[wikilinks]] into the relevant category and detail pages.

The wiki is a drill-down: `index.md` → category page (e.g. `lambdas.md`) →
per-item detail page. Detail pages describe what each thing does in a few
sentences; for specifics like exact file paths or request shapes, read the
source code directly.

Rules:
- Cite wiki page names in answers (e.g. "per [[lambdas/amendArrangement]]").
- If the wiki does not answer a question, say so, then look at the source.
- Do not run `git commit` or `git push` unless explicitly asked.
